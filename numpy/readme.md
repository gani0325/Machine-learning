### numpy
- 파이썬의 고성능 과학 계산용 패키지
- Matrix, Vector 과 같은 Array 연산의 표준
- 일반 list에 비해 빠르고, 메모리 효율적
- 반복문 없이 데이터 배열 처리 지원
- 선형대수 기능 제공

### numpy download
- conda에서
1. 가상환경 만들기
conda createa -n 가상환경이름
2. 가상환경 들어가기
conda activate 가상환경이름
3. numpy 설치하기
conda install numpy
